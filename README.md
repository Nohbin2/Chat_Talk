Chat Talk – 카카오 로그인 기반 실시간 채팅 서비스

🔗 https://github.com/Nohbin2/Chat_Talk

📌 프로젝트 소개

Chat Talk는 카카오 로그인을 통해 사용자 인증을 처리하고, 실시간 채팅 및 이미지 전송 기능을 제공하는 웹 기반 채팅 서비스입니다.
프론트엔드와 백엔드가 분담되어 개발되었으며, 본 저장소는 백엔드 서버 구현 코드를 포함합니다.

🛠 기술 스택

Language: Java

Backend: Spring Boot / MySQL

Authentication: Kakao OAuth 2.0

Real-time Communication: WebSocket (또는 해당 기술)

Data Handling: 메시지/이미지 처리 및 전송

✨ 구현 기능

카카오 로그인 연동

카카오 OAuth API를 활용한 사용자 인증 및 프로필 정보(이름, 사진) 처리

실시간 채팅

사용자 간 메시지 송수신 및 그래픽 스트림 기반 처리

이미지 전송 기능

채팅 내 이미지 업로드 및 실시간 전송 처리

알림 기능

새로운 메시지 수신 시 사용자에게 알림 이벤트 전송

📌 주요 목적

카카오 OAuth 인증 기반 사용자 로그인 구현

실시간 통신 구조 설계 및 메시지/이미지 처리 백엔드 기능 구현

프론트엔드와 API 인터페이스를 맞추며 협업 개발 경험 강화

🚀 특징

별도 회원가입 없이 카카오 계정을 통해 즉시 로그인 및 채팅 기능 이용

WebSocket 기반 실시간 커뮤니케이션

이미지 전송 및 알림 처리 기능 포함
